# CUSTOMER TYPES — PANISSE ÓPTICA

# PERFIL A — PREMIUM

Características:

* busca calidad,
* valora diseño,
* quiere comodidad,
* aprecia exclusividad,
* valora experiencia personalizada.

Estrategia:

* hablar de experiencia,
* adaptación,
* comodidad,
* personalización,
* tecnología,
* estética.

---

# PERFIL B — SENSIBLE A PRECIO

Características:

* pregunta rápido precios,
* compara,
* busca seguridad.

Estrategia:

* claridad,
* honestidad,
* evitar presión,
* explicar diferencias,
* ofrecer tranquilidad.

---

# PERFIL C — MIEDO A PROGRESIVOS

Características:

* malas experiencias previas,
* miedo a mareos,
* inseguridad.

Estrategia:

* transmitir calma,
* explicar adaptación,
* enfatizar personalización,
* explicar diferencias entre lentes.
